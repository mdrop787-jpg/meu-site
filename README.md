# Meu Site Web3 (MVP) com fetch automático de ABI

1. Copie .env.local.example -> .env.local e preencha.
   - Se tiver Polygonscan API key, coloque em POLYGONSCAN_API_KEY.
   - Se o contrato for proxy e você souber o implementation address, preencha IMPLEMENTATION_ADDRESS.
2. npm install
3. npm run dev
4. Abra http://localhost:3000
5. Teste em Mumbai antes de mainnet.
